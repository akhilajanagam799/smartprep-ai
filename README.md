# SmartPrep AI – Interview Preparation Simulator

An AI-powered smart interview preparation simulator built to replicate real-world technical and behavioral interviews. Features include dynamic question generation, voice-based interaction, NLP-based feedback, and performance analytics.

## 💡 Key Features
- Role-based interview question generation (e.g., DSA, DBMS, OS, OOPs, HR)
- Real-time response evaluation using NLP
- Voice-based mock interviews using Web Speech API
- Secure login and user tracking
- Analytics dashboard to monitor progress and improvement

## 🛠 Tech Stack
**Frontend:** React, Tailwind CSS, Recharts  
**Backend:** Node.js, Express.js  
**Database:** MongoDB  
**AI/NLP:** Google Gemini API, Web Speech API  
**Authentication:** JWT, bcrypt
